# 自用VA注释模版


# 函数注释

### 模版内容：

```
 //===============================================================

// @brief: $end$

// @param[in] $MethodArgName$:

// @return: $SymbolType$

//===============================================================
```  

`$MethodArgName$`：函数参数名

`$SymbolType$`：返回值类型
